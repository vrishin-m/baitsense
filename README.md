In order to run this extension:
1. Open Gooogle Chrome
2. Go to manage extensions
3. Toggle on developer mode
4. Select "load unpacked" and locate the folder named "extension" in your download
5. Go to your python interpreter and create a virtual environment
6. Run "pip install -r download.txt"
7. Once this is done, run the python file named "server.py"
8. Now, when you open YouTube and click on a video, you should see a small popup in the bottom right corner, warning you if the video you clicked is clickbait
9. If it doesn't work, try refreshing the extension in the "manage extensions" section
10. Also try turning off your adblocker, as they sometimes cause hindrance to the extension
